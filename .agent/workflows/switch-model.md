---
description: Switch between GLM models and official Anthropic for the entire AI stack
---

To switch the model, run the switch-model script with the desired model ID.

// turbo
1. Run the switch-model script:
`npm run switch-model {{model}}`

Available models:
- `glm-4.7` (Standard)
- `glm-5` (Premium)
- `glm-4.5-air` (Lite)
- `anthropic` (Direct)
