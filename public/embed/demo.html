üì¶ Installation

Add these two lines to your website's HTML (before the closing </body> tag):

<!-- BakedBot Chatbot Embed -->
<link rel="stylesheet" href="https://bakedbot.ai/embed/chatbot.css">
<script>
    // Configure your chatbot
    window.BakedBotConfig = {
        brandId: 'your-brand-id',        // Required: Your BakedBot brand ID
        primaryColor: '#4ade80',         // Optional: Your brand color
        dispensaryId: 'your-disp-id',   // Optional: For dispensary context
        entityName: 'Your Brand Name'   // Optional: For personalization
    };
</script>
<script src="https://bakedbot.ai/embed/chatbot.js"></script>
