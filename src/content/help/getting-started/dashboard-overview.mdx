# Dashboard Overview

Learn to navigate your BakedBot dashboard, understand the interface, and access key features for managing your cannabis business.

## Dashboard Layout

Your dashboard consists of four main areas:

```
┌─────────────────────────────────────────┐
│  [Logo]  Navigation    [Search] [User] │  ← Header
├──────┬──────────────────────────────────┤
│      │                                  │
│ Side │  Main Content Area               │
│ Nav  │  (Changes based on section)      │
│      │                                  │
│      │                                  │
└──────┴──────────────────────────────────┘
```

### 1. Header (Top Bar)

**Left Side:**
- BakedBot logo (click to return home)
- Current organization/brand name

**Center:**
- Global search (find products, customers, orders)

**Right Side:**
- Notifications bell (alerts, updates)
- Help icon (access help center)
- User menu (settings, profile, logout)

### 2. Side Navigation

**Main Sections:**
- 🏠 **Home** - Dashboard overview
- 📦 **Products** - Product catalog
- 🛒 **Orders** - Order management
- 👥 **Customers** - Customer database
- 📊 **Analytics** - Reports and insights
- 📱 **Marketing** - Campaigns and automation
- 🤖 **Agents** - AI agent squad
- ⚙️ **Settings** - Configuration

**Navigation Tips:**
- Click section to expand submenu
- Current page highlighted
- Hover for tooltips
- Collapsible (click hamburger icon)

### 3. Main Content Area

**Displays current section:**
- Dashboard widgets
- Data tables
- Forms and editors
- Reports and charts

**Adaptive:**
- Responsive (works on all screen sizes)
- Context-aware (shows relevant actions)

### 4. Quick Actions Panel (Bottom-Right)

**Floating action buttons:**
- ➕ Create Product
- ✉️ Create Campaign
- 💬 Chat with Smokey
- ❓ Get Help

**Always accessible:**
- From any page
- Quick shortcuts to common tasks

## Loyalty Section

**Dashboard -> Loyalty**

### Loyalty Operations

**Use loyalty tools to:**
- Monitor new members and repeat visitors
- Track rewards adoption and retention performance
- Review contactability before sending campaigns
- Confirm loyalty balances and member activity

### Visitor Check-In Tools

**Dashboard -> Loyalty -> Visitor Check-In QR**
- Download and print the front-door QR code
- Preview the in-store tablet flow
- Preview the public rewards check-in landing page
- Run the staff launch checklist before busy days

**Internal testing:**
- Super users can use the restricted loyalty QA page to validate public check-in, tablet check-in, and expected CRM writes before launch.
